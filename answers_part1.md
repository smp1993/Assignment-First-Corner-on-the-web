# Part 1 — Short Answers

**Q1) What is a hosting provider for web hosting? Give some other ways to host your websites.**  
A hosting provider stores and serves your site’s files from internet-connected servers so users’ browsers can access them. Examples: GitHub Pages, Netlify, Vercel, Cloudflare Pages, Render, Firebase Hosting, or a VPS (e.g., Linux with Nginx/Apache).

**Q2) Does GitHub Pages support custom domain names?**  
Yes. GitHub Pages supports custom domains (e.g., `myname.com`). Official docs: https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site
